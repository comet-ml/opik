Use this guide when you want to validate the Agent Optimizer end-to-end with minimal choices. By the end, you will have a dataset, a metric, and a successful optimization run logged to the Opik dashboard.

## Prerequisites

- Python 3.10+
- An Opik account with an API key (`opik configure`)
- Access to an OpenAI-compatible model via LiteLLM (e.g., `OPENAI_API_KEY`)
- `pipx` or `pip` for installing the SDKs

<Callout>
  Create an account at [comet.com/signup](https://www.comet.com/signup?from=llm) if you have not already. The CLI stores your token in `~/.opik/config`.
</Callout>

## Step-by-step

<Steps>
  <Step title="Install the SDKs">
    ```bash
    pip install --upgrade opik opik-optimizer litellm
    opik configure  # paste your API key when prompted
    ```
  </Step>
  <Step title="Create a dataset and metric">
    ```python
    import opik
    from opik.evaluation.metrics import LevenshteinRatio

    client = opik.Opik()
    dataset = client.get_or_create_dataset(name="agent-opt-quickstart")
    dataset.upsert([
        {"question": "What is Opik?", "answer": "Opik is an LLM observability and optimization platform."},
        {"question": "How do I reduce hallucinations?", "answer": "Use evaluations and prompt optimization to enforce grounding."},
    ])

    def answer_quality(item, output):
        metric = LevenshteinRatio()
        return metric.score(reference=item["answer"], output=output)
    ```
  </Step>
  <Step title="Run the optimizer">
    ```python
    from opik_optimizer import MetaPromptOptimizer, ChatPrompt

    prompt = ChatPrompt(
        messages=[
            {"role": "system", "content": "You are a precise assistant."},
            {"role": "user", "content": "{question}"},
        ],
        model="openai/gpt-4o-mini"
    )

    optimizer = MetaPromptOptimizer(model="openai/gpt-4o")
    result = optimizer.optimize_prompt(
        prompt=prompt,
        dataset=dataset,
        metric=answer_quality,
        max_trials=3,
        n_samples=2,
    )

    result.display()
    ```
  </Step>
  <Step title="Review results in the dashboard">
    - Run `opik dashboard` or open [https://www.comet.com/opik](https://www.comet.com/opik).
    - Filter by the project you set on the `ChatPrompt` (default `Optimization`).
    - Inspect traces, trials, and metric improvements from the run.
  </Step>
</Steps>

## Common issues

| Symptom | Fix |
| --- | --- |
| `ValueError: Prompt must be a ChatPrompt object` | Import `ChatPrompt` from `opik_optimizer` and wrap your messages. |
| `Authentication failed` | Re-run `opik configure` and confirm the API key has Optimizer access. |
| `liteLLM` provider errors | Verify `OPENAI_API_KEY` or provider keys are exported in the shell running the script. |
