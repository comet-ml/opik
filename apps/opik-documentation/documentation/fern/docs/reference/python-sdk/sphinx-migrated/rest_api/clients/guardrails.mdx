# Guardrails Client


The Guardrails client provides methods for managing guardrails in the Opik platform.

> API details for `opik.rest_api.guardrails.client.GuardrailsClient` are available in the generated Core API pages.

## Usage Example


```python

import opik

client = opik.Opik()

# Validate content with guardrails
result = client.rest_client.guardrails.validate(
    checks=[
        {
            "name": "pii_check",
            "enabled": True
        },
        {
            "name": "toxicity_check",
            "enabled": True
        }
    ],
    input="This is the text to validate"
)
```
