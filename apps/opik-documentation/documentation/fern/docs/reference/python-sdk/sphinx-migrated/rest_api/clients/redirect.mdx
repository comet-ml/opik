# Redirect Client


The Redirect client provides methods for handling URL redirects in the Opik platform.

## Usage Example


```python

import opik

client = opik.Opik()

# Handle redirect operations
result = client.rest_client.redirect.redirect(
    target_url="https://example.com/target"
)
```
