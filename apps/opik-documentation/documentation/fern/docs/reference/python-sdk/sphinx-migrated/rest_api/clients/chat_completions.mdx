# Chat Completions Client


The Chat Completions client provides methods for interacting with chat completion endpoints in the Opik platform.

> API details for `opik.rest_api.chat_completions.client.ChatCompletionsClient` are available in the generated Core API pages.

## Usage Example


```python

import opik

client = opik.Opik()

# Create a chat completion
response = client.rest_client.chat_completions.create(
    model="gpt-3.5-turbo",
    messages=[
        {"role": "user", "content": "Hello, how are you?"}
    ],
    temperature=0.7
)
```
