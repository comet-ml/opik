## `opik.Opik`

```python
opik.Opik(
    project_name: str | None = None,
    workspace: str | None = None,
    host: str | None = None,
    api_key: str | None = None,
    _use_batching: bool = False,
    _show_misconfiguration_message: bool = True,
) -> None
```

Initialize an Opik client to log traces and spans.

### Constructor Parameters

- `project_name`: Project name used when logging traces and spans.
- `workspace`: Workspace name (defaults to `default` if not configured).
- `host`: Opik API URL.
- `api_key`: API key (ignored for local deployments).
- `_use_batching`: Internal flag; do not enable unless explicitly needed.
- `_show_misconfiguration_message`: Internal flag for config warning behavior.

### Basic Usage

```python
import opik

client = opik.Opik(project_name="my-project")

trace = client.trace(
    name="hello-trace",
    input={"prompt": "Hello"},
    output={"response": "Hi"},
)
trace.end()

# Flush before process exit to ensure queued events are delivered.
client.flush()
```

## `opik.track`

```python
opik.track(
    name: str | Callable | None = None,
    type: Literal["general", "tool", "llm", "guardrail"] = "general",
    tags: list[str] | None = None,
    metadata: dict[str, Any] | None = None,
    capture_input: bool = True,
    ignore_arguments: list[str] | None = None,
    capture_output: bool = True,
    generations_aggregator: Callable[[list[Any]], Any] | None = None,
    flush: bool = False,
    project_name: str | None = None,
    create_duplicate_root_span: bool = True,
) -> Callable
```

Decorator for function-level tracing.
It supports both `@track` and `@track()` forms.

### Example

```python
import opik

@opik.track
def summarize(text: str) -> str:
    return text[:120]

@opik.track(type="llm", tags=["generation"])
def answer(prompt: str) -> str:
    return "example output"
```

## `opik.configure`

```python
opik.configure(
    api_key: str | None = None,
    workspace: str | None = None,
    url: str | None = None,
    use_local: bool = False,
    force: bool = False,
    automatic_approvals: bool = False,
) -> None
```

Creates or updates local SDK configuration.

### Examples

```python
import opik

# Cloud setup
opik.configure(
    api_key="YOUR_API_KEY",
    workspace="your-workspace",
)

# Local/self-hosted setup
opik.configure(use_local=True)
```
