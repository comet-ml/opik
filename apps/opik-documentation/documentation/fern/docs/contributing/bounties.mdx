# Opik Bounty Program

The Opik Bounty Program is currently **paused until further notice**.

<Tip>
  We are not accepting new bounty claims at this time.
</Tip>

We are keeping this page available so contributors can check status updates in one place.

## Current Status

- New bounty issues are not being published.
- Existing bounty payouts are paused for new work.
- Standard open-source contributions are still welcome through our normal contribution workflow.

## How to Contribute in the Meantime

Please continue contributing through regular GitHub issues and pull requests:

- [Contribution Overview](/contributing/overview)
- [Open Issues on GitHub](https://github.com/comet-ml/opik/issues)
- [Contributor License Agreement (CLA)](https://github.com/comet-ml/opik/blob/main/CLA.md)

If the bounty program resumes, this page will be updated with next steps.
