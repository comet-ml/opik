## Bringing It All Together: Complete LLM Evaluation

This comprehensive video demonstrates the complete [evaluation](/v1/evaluation/overview) workflow in Opik, where [datasets](/v1/evaluation/manage_datasets) and [metrics](/v1/evaluation/metrics/overview) come together to systematically assess LLM performance. You'll see a practical comparison between GPT-4 and [Gemini](/v1/integrations/gemini) models on a RAG application, learn about [prompt versioning](/v1/prompt_engineering/prompt_management), [experiment](/v1/evaluation/evaluate_your_llm) management, and discover how to make data-driven decisions for production deployment. This is where all previous concepts unite into actionable insights.

## Key Highlights

- **End-to-End [Evaluation](/v1/evaluation/overview) Workflow**: Run complete [evaluations](/v1/evaluation/evaluate_your_llm) that process [datasets](/v1/evaluation/manage_datasets), apply models, and score outputs using defined [metrics](/v1/evaluation/metrics/overview) in a systematic pipeline
- **[Prompt Management](/v1/prompt_engineering/prompt_management) & Versioning**: Use Opik's [prompt class](/v1/prompt_engineering/managing_prompts_in_code) to create versioned [prompts](/v1/prompt_engineering/prompt_management) with commit history, ensuring reproducibility and saving time/money
- **Multi-Model Benchmarking**: Compare different models (GPT-4 vs [Gemini](/v1/integrations/gemini)) side-by-side using [evaluation tasks](/v1/evaluation/evaluate_your_llm) and systematic scoring across identical [datasets](/v1/evaluation/manage_datasets)
- **Smart [Experiment](/v1/evaluation/evaluate_your_llm) Organization**: Name [experiments](/v1/evaluation/evaluate_your_llm) strategically (e.g., by model name) for easy identification and comparison rather than relying on random generated names
- **Live [Experiment](/v1/evaluation/evaluate_your_llm) Monitoring**: Track [evaluation](/v1/evaluation/overview) progress in real-time through the Opik UI, viewing [dataset](/v1/evaluation/manage_datasets) processing and results as they're generated
- **Side-by-Side Comparison**: Use the compare feature to evaluate multiple [experiments](/v1/evaluation/evaluate_your_llm) simultaneously, making model selection decisions based on quantitative [metrics](/v1/evaluation/metrics/overview)
- **Template Generation**: Leverage the "Create New Experiment" button to automatically generate [evaluation](/v1/evaluation/overview) scripts with selected [metrics](/v1/evaluation/metrics/overview) for reuse in Python. Each metric in the modal includes a documentation link for quick reference
- **Trace-Level Inspection**: Dive deep into individual responses by opening [traces](/v1/tracing/log_traces) from [experiment](/v1/evaluation/evaluate_your_llm) results to understand model behavior and decision paths
- **Data-Driven Production Decisions**: Choose the best-performing [prompts](/v1/prompt_engineering/prompt_management) and models based on concrete [metrics](/v1/evaluation/metrics/overview) rather than subjective assessment, building confidence for deployment
